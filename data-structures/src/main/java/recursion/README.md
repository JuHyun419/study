### 2진수로 변환하여 출력
  - 음이 아닌 정수 n을 2진수로 변환하여 출력
  - n을 2로 나눈 몫을 먼저 2진수로 변환하여 인쇄(Recursive)
  - n을 2로 나눈 나머지를 인쇄

### Recursion VS Iteration
  - 모든 순환함수는 반복문으로 변경 가능
  - 그 역도 성립. 즉, **모든 반복문은 Recursion 으로 표현 가능**
  - 순환함수는 복잡한 알고리즘을 단순하고 알기쉽게 표현하는 것을 가능하게 함
